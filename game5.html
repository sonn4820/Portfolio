GAMES

RPI Gamfest 2020 Innovation Award Winner

Brotherhood

Itch.IO RPI Gamefest 2020

Brotherhood is the first game I received an award for it, and also the first game I made on the PuzzleScript engine. This is a puzzle game that requires the player needs to control multiple characters at once. Each character can defeat a different type of enemy, so the player needs to wisely use them in order to solve the puzzle (and also the power of each character is hidden and ready for the player to discover). This game had been through many revisions and received much positive feedback.

    My Contributions:
  • Researching and programming the game mechanics
  • Creating 2D pixel characters and enviroment arts
  • Designing the gameplay mechanics
  • Balancing stats and gameplay
